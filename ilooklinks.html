iLookLinks is an iGetter supplemental app that adds the following features:

  • Lists completed by iGetter downloads and allows you to open VirusTotal recent report for selected file's MD5 checksum. You need to comply to VirusTotal Terms of Service if you want to use this functionality. For more info see.
  • Includes an integrated browser with a local cookie repository along with support for Safari, Firefox and Chrome cookies. This allows you to copy&paste or drag&drop (on the Dock Icon) download links from your browser to iLookLinks, which in turn will process them, attach proper cookies and send the links to iGetter. If some error occurs while processing links, it will show a web page where you can login and proceed.
  • Add links to iGetter running on another Mac. Application may act as a server (see Preferences > Advanced panel), so any link(s) dropped on the local application will be processed on the remote machine. You may select a remote machine (or type its IP address and port) in the remote-address combobox on main window (only LAN machines with started iLookLinks in server mode will appear there). You can use SSH tunnel to send links over the Internet to your home desktop machine for example.
